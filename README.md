# Kambaz

CS4550 Web Development - Assignment 1

A Learning Management System prototype built with Next.js.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the application.

## Project Structure

- `/app/labs` - HTML examples and labs
- `/app/(kambaz)` - Kambaz LMS application
  - `/account` - Sign in, Sign up, Profile screens
  - `/dashboard` - Course dashboard
  - `/courses/[cid]` - Course pages (Modules, Assignments, etc.)
